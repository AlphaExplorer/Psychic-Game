GOT Hangman Style

"Chaos is a ladder"-Littlefinger. Chaos is indeed the name of this game and your goal is to create as much of it as possible. Between you and the Iron Throne there is a list of characters you need to eliminate. In order to do so, you need to guess within the allowable attempts the name and last name of your victim. Be careful, because if you lose, you will have to start from the beginning.

Press any key to get started
